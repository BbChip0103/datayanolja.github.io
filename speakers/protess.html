실시간 빅데이터 분석을 위한 삼위일체를 소개합니다


발표자 : 독고혁 ( )

시간 : 14:30 ~ 15:10 (A)

난이도 : 중간

키워드 : 실시간 데이터 분석, 인메모리 컴퓨팅, Apache Ignite, Apache Flink, Apache Kafka

내용

스트리밍 데이터는 실시간으로 쏟아져 들어오는 빅데이터를 실시간으로 가공 및 분석하여 의미있는 데이터로 변환 하는것을 목표로 합니다.

가장 좋은 예로 온라인 쇼핑몰, 넷플릭스, 유투브 등에서 자주 접하게 되는 추천 상품과 추천 동영상들은 배치가 아닌 실시간으로 이루어 지고 있으며 이젠 스트리밍 데이터 분석은 필요가 아닌 필수가 되었다고 해도 과언이 아니라고 생각됩니다

이번 발표에서 어려운 스트리밍 데이터에 대한 개념이나 상세한 내용을 이야기하는것이 아니라 실시간으로 데이터를 분석하기 위한 플랫폼들에 대한 소개를 드리고 왜 Kafka + FLink + Ignite 를 선택했는지에 대한 내용을 공유 드리고자 합니다.

대상 청중

실시간 데이터를 해결해야 하는 과제에 놓여 있는데 어떤 플랫폼을 써야하는지 도움이 필요하거나 새로운 기술에 관심이 많으신분.

발표 이해에 도움 되는 사전 지식

Modern Java, In-Memory Computing, Streaming Data, Distributed Clustering
